# Clasificación de Vinos - Taller Machine Learning

## Descripción

Proyecto desarrollado como parte del taller de Machine Learning. El objetivo es construir un modelo de clasificación capaz de predecir la categoría de vinos basándose en características físico-químicas.

## Contenido

Este repositorio contiene el código y análisis necesario para:
- Exploración y procesamiento de datos de vinos
- Entrenamiento de modelos de clasificación
- Evaluación del desempeño predictivo

## Tecnología

- **Lenguaje**: Python
- **Propósito**: Aplicación práctica de técnicas supervisadas de machine learning

## Autor

Desarrollado como actividad del taller de ML.
